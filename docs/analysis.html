3 Analysis

Define no_ias_records as and ordered factor:

input_data$no_ias_records <- factor(input_data $ no_ias_records, ordered = TRUE)

Attempt for analysing the data - Independent variable: number of invasive alien records - Dependent variables: - duration - social media - species id - guidlines - training - sightings map - active informating - feedback

model <- 
  clm(
    no_ias_records ~ duration + design + social_media + species_id + guidelines + training + sightings_map + active_informing + feedback,
    data = input_data)
summary(model)
